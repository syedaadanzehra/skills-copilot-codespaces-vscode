# Contributing

Welcome! 👋 This repo is a [GitHub Skills](https://skills.github.com/) exercise for learning to **code with GitHub Copilot**. Here's how to work on it and complete the steps.

## Getting started

1. **Use this template** to create your own copy of the repo (don't clone this one directly).
2. Open the repo on GitHub and read the `README.md` — it's the exercise. Each step appears there one at a time.
3. Follow the instructions in the current step, then **wait ~60 seconds and refresh** the repo landing page to reveal the next step.

## Completing the exercise

- The exercise runs best inside a **Codespace** with the Copilot extension enabled (Step 1 walks you through this).
- Make the changes each step asks for (creating files, committing, etc.) and **commit directly to `main`** unless told otherwise.
- An automated workflow checks your work and advances the exercise — give it a moment after each commit.

## Tips

- Open the instructions in one browser tab and do the work in another.
- Stuck? [Post in the discussion board](https://github.com/orgs/skills/discussions/categories/code-with-copilot).
- Take your time — there's no rush, and you can redo steps if needed.

Happy coding! 🚀
