# NOTES

This repository is an interactive **GitHub Skills course** titled "Code with GitHub Copilot," built from GitHub's Skills course template. Rather than containing application source code, it's a guided, step-by-step tutorial (driven by the README and GitHub Actions in `.github/`) that teaches learners how to enable and use GitHub Copilot's AI-powered code suggestions inside a Codespace with VS Code — starting with creating a dev container that installs the Copilot extension.
